নোটিফিকেশন সিস্টেম অ্যাডমিন প্যানেল

GitHub রিপোজিটরি ব্যবহার করে নোটিফিকেশন ব্যবস্থাপনা

এই টোকেন কোথাও সেভ করা হবে না, শুধুমাত্র বর্তমান সেশনে ব্যবহার করা হবে

প্রিভিউ:

ব্যবহার নির্দেশিকা:

  1. আপনার GitHub অ্যাকাউন্ট থেকে একটি Personal Access Token তৈরি করুন
  2. উপরে টোকেন এবং রিপোজিটরি URL ইনপুট করুন
  3. নোটিফিকেশন বার্তা লিখুন (HTML ট্যাগ ব্যবহার করতে পারেন)
  4. "পাবলিশ করুন" বাটনে ক্লিক করুন নোটিফিকেশন সক্রিয় করতে
  5. "নিষ্ক্রিয় করুন" বাটনে ক্লিক করুন নোটিফিকেশন বন্ধ করতে
  6. আপনার ওয়েবসাইটে notification.js ফাইলটি যুক্ত করুন
